# Verschieben, umbenennen

Um Dateien zu verschieben oder umzubenennen, nutzen wir das Kommando `mv`. Als erstes Argument geben wir den Namen der Datei mit, welche verschoben bzw. umbenannt werden soll. Das zweite Argument ist der neue Name der Datei, oder das Verzeichnis in welches es verschoben werden sollte:

````Bash
mv helloWorld.sh ../Documents
mv script.sh script2.sh
````
